# Library Management System (C++)
A simple console-based library management system developed in C++ as a group project.

## About the Project
This project was created by a team of three students as part of their major assignment. It implements core OOP concepts through a functional library interface.

## Features
- **Login System** – Authenticate using name and ID
- **Add Book** – Input book title, author, and ID
- **Show Books** – View all stored books
- **Search Book** – Find book by ID with retry option
- **Sort by ID** – Sort books using bubble sort
- **Menu-Driven** – Easy navigation with numbered options
