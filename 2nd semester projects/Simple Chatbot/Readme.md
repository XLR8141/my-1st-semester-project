# Mock Interview Chatbot (Python + Tkinter)
A simple **GUI-based mock interview practice tool** built with Python and Tkinter.

## About the Project
This desktop application simulates a chat-based interview session. Users can practice **coding** or **system design** questions with instant feedback. Ideal for technical interview preparation.

## Features
- **Interactive Chat Interface** – Clean, modern dark theme
- **Two Modes**:
  - coding – Algorithm & data structure questions
  - design – System design questions
- **Smart Feedback** – Checks for relevant keywords in answers
- **Next Question Flow** – Automatically proceeds after each response
- **Commands**:
  - coding Start coding questions
  - design Start design questions
  - restart Reset session
